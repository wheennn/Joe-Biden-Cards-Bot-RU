# Joe Biden Cards Bot [RU]

Joe Biden Cards – это коллекционный телеграм-бот с элементами соревнования и охоты за наградами. Здесь вы можете поохотиться за редкими картами, посоревноваться за хорошее место в списке лидеров или просто убить время в ожидании заказа.

**v0.1.1.3 - 02.06.26 (технический патч)**

– Исправлена ошибка в фильтре «по первенству» списка лидеров, из-за которой данные отображались некорректно.

– Оптимизация базы данных для повышения стабильности работы бота.

– Внутренний административный инструмент для вывода и интерпретации данных игроков из базы данных.

– Изменен алгоритм подсчета общего количества карт в списке лидеров.

– Скорректированы тайминги напоминаний, устранена системная ошибка ложного срабатывания при активном времени перезарядки.

– Распознавание более широкого спектра ответов от пользователей: для подтверждения или отмены открытия теперь могут использоваться русские буквы вместо английских и в выборе фильтра укороченные версии, например "по" вместо по опыту и др.

## **Для успешного запуска бота следуйте следующей инструкции:**

1. Скачайте все файлы, прикрепленные к репозиторию, на свой компьютер

2. Установите библиотеки aiogram и python-dotenv, если они у вас еще не скачаны:

    Скопируйте `pip install -U aiogram python-dotenv` и вставьте в командную строку (Win + R чтобы открыть, дальше введите cmd)

3. Поскольку оригинальный файл data.env скрыт в целях безопасности, создайте собственный файл, подставив другие значения:
```env
BOT_TOKEN=ваш_токен (скопируйте его, создав бота в @BotFather и вставьте здесь)
DATABASE_v0.1.1.3=database_v0.1.1.3.db (по умолчанию, если хотите, можете заменить название файла на свое)
DEV_ID=ваш_айди (вы можете получить его в @userinfobot)
```
4. Запустите бота через командную строку по примеру ниже:
```bash
python "Joe Biden Cards v0.1.1.3.py"
```


**Обратите внимание:** Проект находится на стадии активной разработки в Beta, поэтому он может включать в себя баги, временные сбои или ощущение «сырости» некоторых механик.
