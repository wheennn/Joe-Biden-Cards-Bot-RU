# Joe Biden Cards Bot [RU]

## Про бота

Joe Biden Cards – это коллекционный телеграм-бот с элементами соревнования и охоты за наградами. Здесь вы можете поохотиться за редкими картами, посоревноваться за хорошее место в списке лидеров или просто убить время в ожидании заказа.

Проект находится в бета-версии, поэтому он может включать в себя баги, временные сбои или ощущение «сырости» некоторых механик.

Оригинальный бот в Telegram: @joebidencardbot

Язык: Русский

## Установка

**Следуйте следующей инструкции, чтобы успешно скачать и запустить бота:**

1. Скачайте все файлы, прикрепленные к репозиторию, на свой компьютер

2. Установите библиотеки aiogram и python-dotenv, если они у вас еще не скачаны:

    Скопируйте `pip install -U aiogram python-dotenv` и вставьте в командную строку (Win + R чтобы открыть, дальше введите cmd)

3. Поскольку оригинальный файл data.env скрыт в целях безопасности, создайте собственный файл, подставив другие значения:
```env
BOT_TOKEN=ваш_токен (Токен вашего бота. Получите его в Telegram у официального бота @BotFather, создав нового бота через команду /newbot)
DATABASE=database.db (Имя файла базы данных SQLite. Можно оставить database.db без изменений)
USERS_WITH_FULL_COLLECTION=айди_игрока1,айди_игрока2,айди_игрока3,айди_игрока4 (ID пользователей, которые открыли полную коллекцию до добавления новых карт. Если таких нет, – введите любые случайные числа)
DEV_ID=ваше_айди (Ваш личный цифровой ID в Telegram. Узнать его можно через ботов @userinfobot или @MyIdBot)
DEV2_ID=айди_второго_разработчика (ID второго разработчика. Если вы делаете проект в одиночку, просто скопируйте сюда значение из DEV_ID)
DEV_MINI_ID=айди_вашего_твинка (ID вашего второго аккаунта Telegram. Если твинка нет, продублируйте сюда значение из DEV_ID)
TOP1_SEASON0=топ1_досезона (ID игрока, который занял первое место в нулевом сезоне. Если проект новый, напишите любое число)
COMPENSATION1=пользователь1 (ID первого игрока, которому после запуска будет выдана награда за уже достигнутые глобальные достижения)
COMPENSATION2=пользователь2 (ID второго игрока, которому после запуска будет выдана награда за уже достигнутые глобальные достижения)
COMPENSATION3=пользователь3 (ID третьего игрока, которому после запуска будет выдана награда за уже достигнутые глобальные достижения)
COMPENSATION4=пользователь4 (ID четвёртого игрока, которому после запуска будет выдана награда за уже достигнутые глобальные достижения)
COMPENSATION5=пользователь5 (ID пятого игрока, которому после запуска будет выдана награда за уже достигнутые глобальные достижения)
COMPENSATION6=пользователь6 (ID шестого игрока, которому после запуска будет выдана награда за уже достигнутые глобальные достижения)
```
4. Запустите бота через командную строку:
```bash
python "main.py"
```
